# Canonical Style Spec

Дата фиксации: 27 марта 2026

Цель: единый продуктовый канон для deterministic-слоя, prompt и benchmark-разметки.

## Standard

- Деньги: `900 рублей`
- Деньги с копейками: `900 рублей 50 копеек`
- Разделитель тысяч: `20 000 рублей`
- Проценты: `25%`
- Время: `15:30`
- Неоднозначное время без контекста: `в пять`
- Явное время суток: `в девять вечера`
- Дата: `23 марта 2026`
- Дата и время: `23 марта 2026 в 15:30`
- Единицы измерения: полные формы, например `5 килограммов`, `2 литра`, `10 километров`
- Температура: `25°C`
- Бренды и сервисы: официальное написание, например `Jira`, `Slack`, `Notion`, `GitHub`, `Telegram`, `Zoom`
- Техтермины: `PDF`, `CSV`, `iOS`, `QA`, `ML`
- Идентификаторы и служебные токены: точное написание, например `project.yml`, `MARKETING_VERSION`, `CURRENT_PROJECT_VERSION`, `Sparkle`
- Самокоррекция: сохраняется только финальная версия
- Пунктуация: нормальная письменная
- Конечная точка по умолчанию: выключена
- При конфликте важнее единый канон, чем буквальное воспроизведение формы устной речи

## Owner-правила

- Deterministic слой владеет числами, валютами, датами, временем, единицами, брендами и техтерминами.
- LLM не меняет уже нормализованные canonical forms.
- Benchmark full-pipeline должен мерить именно этот канон, а не старые `llm-only` ожидания.
