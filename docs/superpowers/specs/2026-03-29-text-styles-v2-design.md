# Стили текста v2

Замена шести контекстных режимов (chat, email, document, note, code, universal) на три уровня формальности с глобальным переключателем авто/ручной.

## Три стиля

| | Расслабленный | Обычный | Формальный |
|---|---|---|---|
| Капитализация | строчная | С заглавной | С заглавной |
| Точка в конце | нет | нет | да |
| Обращение | ты | ты | Вы |
| Бренды | кириллица (слак, зум) | оригинал (Slack, Zoom) | оригинал (Slack, Zoom) |
| Сленг (норм, спс) | сохранять | сохранять | раскрывать (нормально, спасибо) |
| Филлеры (ну, короче) | убирать | убирать | убирать |

## Режим выбора: Авто vs Ручной

Глобальный переключатель:

- **Авто** — стиль определяется по активному приложению:
  - Расслабленный: Telegram, WhatsApp, Viber, VK Messenger, iMessage, Discord
  - Формальный: Mail, Spark, Outlook
  - Обычный: всё остальное (Яндекс Браузер, Safari, Chrome, Arc, Notes, Notion, Obsidian, Xcode, VS Code, JetBrains, Pages, МойОфис, Word)
- **Ручной** — пользователь выбирает один стиль, он применяется ко всем приложениям

Per-app overrides удаляются.

## Миграция

- Enum `TextMode`: 6 кейсов → 3 (`relaxed`, `normal`, `formal`)
- UserDefaults: `chat` → `relaxed`, `email`/`document` → `formal`, `note`/`code`/`universal` → `normal`
- `AppContextEngine` — дефолтный маппинг обновляется на новые bundleId (VK Messenger, Viber, Яндекс Браузер, МойОфис)
- `AppModeOverriding` протокол и `UserDefaultsAppModeOverrides` — удаляются
- `AppModeSettingsView` — удаляется
- `SettingsStore` — новое поле `styleSelectionMode: .auto | .manual` + `manualStyle: TextMode`
- Стиль-блоки в промпте перегенерируются под три режима
- Seed dataset (36 сэмплов) — пересмотреть expected-значения

## Тестирование

- Unit-тесты `AppContextEngine`: обновить на 3 стиля, добавить новые bundleId
- Unit-тесты `TextMode`: миграция старых rawValue, styleBlock для каждого стиля
- Unit-тесты `SettingsStore`: переключение auto/manual, персистенция manualStyle
- UI-тесты: состояние вкладки Супер при наличии/отсутствии модели, переключение сегмента, выбор стиля
- Seed dataset: прогнать бенчмарк после обновления промпта
