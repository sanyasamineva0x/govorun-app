## Formal slang vocabulary

| Option | Description | Selected |
|--------|-------------|----------|
| Фиксированная таблица как в styleBlock | Те же пары, что LLM получает в промпте (норм→нормально, спс→спасибо, ок→хорошо) как static let. Gate использует для валидации | ✓ (расширено) |
| Ослабить порог для formal | Без таблицы — поднять порог edit distance для formal. Проще, менее точно | |
| Игнорировать сленг в distance | Удалить сленговые токены при distance. Без таблицы, эвристика | |

**User's choice:** Фиксированная таблица, но расширенная до ~15-20 пар (не только 3 из styleBlock)
**Notes:** Пользователь обеспокоен, что 3 пары из styleBlock недостаточно для реального русского сленга. Решение: расширить до ~15-20 частых пар (чё→что, щас→сейчас, инфа→информация, комп→компьютер, прога→программа, чел→человек и др.)

## Незнакомые бренды/сленг (дополнительный вопрос от пользователя)

| Option | Description | Selected |
|--------|-------------|----------|
| Оставить как есть | STT даёт кириллицу, латиница редкость. Таблицу можно расширять потом. Осознанный компромисс | ✓ |
| Ослабить protected check в relaxed | Если Latin token отсутствует и не в таблице — не отклонять, полагаться на distance | |
| Добавить больше алиасов | Расширить таблицу сейчас (scope creep — спека фиксирует 25+4) | |

**User's choice:** Оставить как есть — осознанный компромисс
**Notes:** Пользователь поднял проблему: таблицы алиасов конечны, незнакомые бренды/сленг не покрыты. Для брендов: STT даёт кириллицу, латиница во входе редкость. Для сленга: расширенная таблица (~15-20 пар) покроет 90% случаев. Общий ослабленный порог — запас для остального.
