    **Change 4: missingProtectedTokens() -- alias-aware** (per GATE-02, GATE-04, D-06, D-07)
    Add `superStyle: SuperTextStyle?` parameter. Logic:

    ```swift
    private static func missingProtectedTokens(
        expected: [String],
        actualText: String,
        ignoredOutputLiterals: Set<String>,
        superStyle: SuperTextStyle?
    ) -> [String] {
        let actualCanonical = Set(
            extractProtectedTokens(from: actualText, ignoredLiterals: ignoredOutputLiterals)
        )
        let lookup = aliasLookup(for: superStyle)

        return expected.filter { token in
            let canon = canonicalize(token)
            // Direct match
            if actualCanonical.contains(canon) { return false }
            // Alias match: token's canonical maps to some canonical form; check if that form is in output
            if let canonical = lookup[canon], actualCanonical.contains(canonical) { return false }
            // Reverse: maybe token IS the canonical and output has the variant
            // Check all entries that map to this token's canonical form
            for (variant, target) in lookup where target == canon {
                if actualCanonical.contains(variant) { return false }
            }
            return true
        }
    }
    ```

    Note: The "reverse" loop handles the case where input has the canonical form (e.g., "Slack") and output has the variant (e.g., "слак"). The lookup has слак->slack, but we need slack->слак direction too. The loop checks if any variant maps to the same canonical as our token.

    OPTIMIZATION NOTE: The reverse loop is O(n) per token. For 29 relaxed entries and 17 formal entries this is negligible. If performance becomes a concern later, build a reverse dictionary. Do not over-engineer now.

    **Change 5: Style-neutral distance normalization** (per GATE-03, D-03, D-04)
    Add a new private function and integrate into the distance flow:

    ```swift
    private static func normalizeStyleTokens(
        _ tokens: [String],
        style: SuperTextStyle?
    ) -> [String] {
        let lookup = aliasLookup(for: style)
        guard !lookup.isEmpty else { return tokens }
        return tokens.flatMap { token -> [String] in
            if let canonical = lookup[token] {
                // Split canonical form into words (handles "мб" -> "может быть")
                let parts = canonical.split(whereSeparator: \.isWhitespace).map(String.init)
                return parts.isEmpty ? [token] : parts
            }
            return [token]
        }
    }
    ```
